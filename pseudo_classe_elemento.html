Trabalhando com Pseudo-classe

São usadas para definir um elemento com um estado especial

Serve para:

  • Estilizar um elemento quando um usuario passa o mouse sobre ele
  • Estilizar o link visitado e o não visitado de forma diferente
  • Estilizar um elemento quando le recebe o foco

PSEUDO-CLASSE

pseudo Elemento

Um Pseudo-elemento é usado para estilizar partes específicas de um elemento

Por exemplo:

  • Estilize a primeira letra ou linha de um elemento.
  • Insira conteúdo antes ou depois do conteúdo de um elemento.

NOTA:

O ::first-line só pode ser aplicado a um elemento de nível de bloco.

As seguintes propriedades se aplicam ao ::first-line:

  • Propriedade da fonte.
  • Propriedade da cor.
  • Propriedade de fundo.
  • Espaçamento entre as palavras.
  • Espaçamento entre as letras.
  • Decoração do texto.
  • Alinhamento vertical.
  • Transformação de texto.
  • Altura da linha.
